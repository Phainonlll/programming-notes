## 1.1取地址运算：&运算符取得地址变量

①sizeof:
-是一个运算符，给出某个类型或变量在内存中所占据的字节数
-sizeof(int)~int占了几个字节
-sizeof(i)~i占了几个字节

② %p 是C/C++语言中用于格式化输入或输出的格式说明符，专门用于打印指针（内存地址）的值

③一个字节8个比特

## 1.2指针：指针变量就是记录地址的变量

①指针：就是保存地址的变量
-int i;
-int*p= & i;  （表示:p是一个指向i的int类型指针，p获得i的指针）
-int*p,q;      （ *可以靠近int也可以靠近指针
-int *p,q;      q不是指针）

② 作为参数的指针
-void f(int *p);   //声明了一个f函数                    void f(int *p);
                                                        int main(void)
                                                        {
                                                            int i=6;
                                                            printf("&i=%p\n",&i)
                                                            f(&i);
                                                            return 0;
                                                        }
                                                        void f(int *p)
                                                        {
                                                            printf("p=%p\n",p)
                                                            printf("*p=%d\n,*p) //*p为一个整体,是i的地址
                                                            *p=26       //让i地址为26，i被重新赋值为26了
                                                        }
                                                        运行结果：
                                                        &i=0xbff17d70
                                                        p=0xbff17d70
-在被调用时得到了某个变量的地址
-int i=0；f(&i);    //此处调用f函数
-在函数里可以通过这个指针访问外面这个

③访问那个地址的变量 *
-* 是一个单目运算符，用来访问指针的值所表达的地址上的变量
-可以做左值也可以做右值
-int k=*p;
-*p=k+1;

## 1.3 指针与数组

①
-函数参数表中的数组实际上是指针     p[n]等价于*(p+n)
                                    p[-2]即*(p-2) 其含义为从指针p所指向的位置，向前移动两个元素位置，后取该位置上的值
-sizeof(a)==sizeof(int*)
-但是可以用数组的运算符[]进行运算

②数组参数
-以下四种函数原型是等价的
  -int sum(int*ar,int n);           -int sum(int*,int);
   int sum(int ar[],int n);         int sum(int[,int]);

③数组变量是特殊的指针
    -数组变量本身表达地址，所以
        -int a[10];int*p=a;     //无需用&取地址
        -但数组单元表达的是变量，需用&取地址
        -a==&a[0]
    -[]运算符可以对数组做，也可以对指针做
        -p[0]<==>a[0]
    -*运算符可以对指针做，也可以对数组做
        -*a=25
    -数组变量是const的指针，所以不能被赋值
        -int a[]<==>int* const a=...


## 2.1字符类型

①字符类型
-char是一种整数，也是一种特殊的类型：字符  //character
-用单引号表示字符字面量：'a','1'
-"也是一个字符
-printf和scanf里用%c来输入输出字符
eg：{
    char c;
    c=1;            c=1
    char d;
    d='1';          d=49(ASCII)
}

②字符的输入与输出
-如何输入'1'这个字符给char c?
    -scanf("%c",&c);    -->1
    -scanf("%d",&i);  c=i;   -->49

eg:{
    char c;                 //输入：
    scanf("%c",&c);                 1
    printf("c=%d\n",c);     //输出：c=49
    printf("c='%c'\n",c);             c='1'
}
-'1'的ASCII的编码是49，所以当c==49,它代表'1'
    -printf("%i %c\n",c,c);
    -一个49各自表述：-当整数：49
                    -当字符：'1'

③字符计算
char c='A';
c++;
printf("%c\n",c);
-一个字符加一个数字得到ASCII编码表中那个数之后的字符
int i='z'-'A';
printf("%d\n",i);
-两个字符的减，得到它们在表中的距离

④大小写转换
-字母在ASCII表中是顺序排列的
-大写字母和小写字母是分开排列的
-'a'-'A'可以得到两端的距离，于是：
    -a+'a'-'A'可以把一个大写字母变成一个小写字母
    -a+'A'-'a'可以把一个小写变成一个大写。

## 3.1字符串

①char word[]={'H','e','l','l','o','!','\0'};
word[0]    H
word[1]    e
word[2]    l
...
word[5]    !
word[6]    \0

②-以0（整数0）结尾的一串数字
-0或'\0'是等价的，但不同于字符'0'(ASCII值为48)
-\0表示字符串的结束，并不计算在字符串长度内
-字符串以数组的形式存在，以数组或指针的形式访问，更多是以指针的形式
-string.h里有很多处理字符串的函数

③字符串变量
-char *str="Hello";     指针指向字符Hello
-char world[]="Hello";      字符数组world里的内容是"Hello"
-char line[10]="Hello";

④字符串字面量/常量
-"Hello"    由双引号括起来的东西
-"Hello"会被编译器变成一个字符数组放在某处，长度是6，结尾还有表示结束的0
-两个相邻的字符串常量会被自动连接起来。

## 3.2字符串变量
-字符数组（最常用）     char str[]="Hello";
                       str[0]='h';      //可修改
-动态分配的字符串       char *str=malloc(100); 
                       strcpy(str,"Helllo");
                       str[0]='h';    //可修改

①字符串常量
char* s="Hello,world";
-s是一个指针，初始化为指向一个字符串常量
-由于这个常量所在的地方，所以实际上s是从上const char* s，
但是由于历史原因，编译器接受不带const的写法
-char *s="Hello,World!"是只可读不可写的
-若要修改字符串，须用数组： char s[]="Hello,World!";

②该如何选择：char *str="Hello";?    或  char word[]="Hello";?
-数组：这个字符串在这里
      -作为本地变量空间自动被回收       //若要构造一个字符串 -->数组
-指针：这个字符串不知道在哪里             //处理          -->指针
      -处理参数
      -动态分配空间

③char* 是字符串？
-字符串可以表达为char* 的形式
-char* 不一定是字符串
      -本意指向字符的指针，可能指向的是字符的数组（就像int* 一样）
      -只有它所指的字符数组有结尾的0，才能说它所指的是字符串。
