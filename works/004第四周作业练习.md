## 1.奇偶个数
题目内容：

你的程序要读入一系列正整数数据，输入-1表示输入结束，-1本身不是输入的数据。程序输出读到的数据中的奇数和偶数的个数。



输入格式:

一系列正整数，整数的范围是（0,100000）。如果输入-1则表示输入结束。



输出格式：

两个整数，第一个整数表示读入数据中的奇数的个数，第二个整数表示读入数据中的偶数的个数。两个整数之间以空格分隔。



输入样例：
9 3 4 2 5 7 －1 



输出样例：
4 2

C

🪻提交如下：

#include <stdio.h>

int main() {
    int number;
    int odd = 0, even = 0;
    
    scanf("%d", &number);
    
    while(number != -1) {
        if(number % 2 == 0) {
            even++;
        } else {
            odd++;
        }
        scanf("%d", &number);
    }
    
    printf("%d %d\n", odd, even);
    
    return 0;
}




## 2.数字特征值
题目内容：

对数字求特征值是常用的编码算法，奇偶特征是一种简单的特征值。对于一个整数，从个位开始对每一位数字编号，个位是1号，十位是2号，以此类推。这个整数在第n位上的数字记作x，如果x和n的奇偶性相同，则记下一个1，否则记下一个0。按照整数的顺序把对应位的表示奇偶性的0和1都记录下来，就形成了一个二进制数字。比如，对于342315，这个二进制数字就是001101。

程序要读入一个非负整数，整数的范围是[0,1000000]，然后按照上述算法计算出表示奇偶性的那个二进制数字，输出它对应的十进制值。



提示：将整数从右向左分解，数位每次加1，而二进制值每次乘2。



输入格式:

一个非负整数，整数的范围是[0,1000000]。



输出格式：

一个整数，表示计算结果。



输入样例：

342315



输出样例：

13

C


🪻提交如下：

#include <stdio.h>

int main() {
    int num;
    scanf("%d", &num);
    
    int n = 1;
    int result = 0;
    int base = 1;
    
    while (num > 0) {
        int digit = num % 10;
        
        if ((digit % 2) == (n % 2)) {
            result += base;
        }
        
        base *= 2;
        n++;
        num /= 10;
    }
    
    printf("%d\n", result);
    return 0;
}
